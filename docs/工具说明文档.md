## 一、作品简介

本作品是一个用于辅助理解“方向导数与梯度”的交互式网页。网页把多元函数 `z=f(x,y)` 的公式、三维曲面图像、选定点、方向向量、梯度方向和方向导数结果放在同一个界面中，让学生能够一边输入、一边观察、一边验证。

方向导数和梯度是数学分析中比较抽象的内容。学生在学习时常常会遇到这样的问题：公式可以代入计算，但很难真正看出“沿某个方向变化得快不快”“梯度为什么表示增长最快方向”“方向向量为什么要单位化”。本网页希望通过可拖拽的三维曲面和动态箭头，把这些概念变得更直观。

## 二、作品解决的问题

### 1. 解决“只会算，不会看”的问题

传统学习中，方向导数往往以公式形式出现：

```text
D_u f(P0) = ∇f(P0) · u
```

学生可以按照公式求偏导、算点乘，但不一定能理解这个数值在图像上的含义。本网页把计算结果和曲面上的点、方向箭头、梯度箭头同步展示，帮助学生把“公式结果”对应到“曲面上的斜率变化”。

### 2. 解决“方向感不强”的问题

方向导数的关键在于“方向”。同一个点，沿不同方向移动，函数值变化率可能不同。本网页允许用户输入方向向量，也可以通过方向角滑条改变方向，从而观察方向改变后方向导数如何变化。

### 3. 解决“梯度概念抽象”的问题

梯度不仅是两个偏导数组成的向量，它还表示函数增长最快的方向。网页会在图像中标记梯度方向，并与用户选定的方向作对比，帮助学生理解方向导数本质上是梯度在指定单位方向上的投影。

### 4. 解决“函数图像不容易想象”的问题

对于 `x^2+y^2`、`x^2-y^2`、`sin(x)cos(y)` 等多元函数，学生仅凭公式很难想象曲面形状。本网页提供三维函数图像，支持鼠标拖拽旋转、缩放和平移，可以从不同角度观察曲面。

## 三、核心功能介绍

### 1. 函数输入与图像生成

用户可以输入显式曲面函数：

```text
z = f(x,y)
```

例如：

```text
z = x^2 + y^2
z = x^2 - y^2
z = sin(x)cos(y)
```

输入函数后，网页会自动在三维坐标系中生成函数曲面。图像可以通过鼠标拖拽从不同方向观察。

### 2. 常见函数预设

为了降低使用门槛，网页内置了几个适合课堂讲解的函数例子：

| 预设函数 | 适合理解的内容 |
| --- | --- |
| 抛物面 `z=x^2+y^2` | 验证梯度与方向导数的基础关系 |
| 马鞍面 `z=x^2-y^2` | 观察同一点不同方向斜率一正一负 |
| 波纹面 `z=sin(x)cos(y)` | 观察周期性曲面上的方向变化 |
| 指数丘陵 | 观察向峰顶移动时函数值上升 |
| 对数山谷 | 观察缓慢增长的曲面 |
| 径向距离 | 理解距离函数的径向变化 |

点击预设后，网页会自动填入函数、推荐点和推荐方向，适合课堂快速演示。

### 3. 点与方向向量输入

用户可以输入点的平面坐标：

```text
P0 = (x0, y0)
```

网页会自动把点吸附到曲面上，即自动计算：

```text
z0 = f(x0,y0)
```

用户还可以输入方向向量：

```text
v = (vx, vy)
```

网页会自动将方向向量单位化，再用于方向导数计算。

### 4. 方向导数计算与展示

网页会展示：

- 当前点在曲面上的位置；
- 该点处的梯度；
- 单位方向向量；
- 对应方向导数的数值结果；
- 代入后的计算公式。

例如在函数：

```text
z = x^2 + y^2
```

取点：

```text
P0 = (1, 1)
```

取方向：

```text
v = (1, 0)
```

网页会得到：

```text
∇f(1,1) = (2,2)
u = (1,0)
D_u f(P0) = 2
```

这个结果说明：在点 `(1,1)` 处沿 `x` 正方向移动时，函数值的瞬时变化率为 `2`。

### 5. 图像标记辅助理解

网页会在三维曲面上标记：

- 选定点；
- 用户选定方向对应的箭头；
- 梯度方向箭头；
- 沿指定方向经过该点的曲面截线；
- 底面参考方向线。

这些标记可以帮助学生直观看到“点在哪里”“往哪个方向走”“梯度指向哪里”“沿这个方向函数值如何变化”。

## 四、网页优点

### 1. 数形结合，降低抽象难度

方向导数本身是公式化概念，但其几何意义非常重要。网页把公式、图像和结果放在一起展示，使学生不仅能算出答案，也能看到答案对应的几何意义。

### 2. 操作简单，适合课堂展示

网页不需要安装 GeoGebra、Mathematica 等软件，只需打开链接即可使用。预设函数可以一键切换，适合老师课堂讲解，也适合学生汇报演示。

### 3. 支持自主探索

学生可以自由改变函数、点和方向，观察结果变化。相比静态课件，这种交互方式更容易激发学生主动思考，例如：

- 如果方向和梯度一致，方向导数会怎样？
- 如果方向和梯度垂直，方向导数会怎样？
- 马鞍面上为什么有些方向上升，有些方向下降？

### 4. 自动处理单位方向

方向导数定义中要求方向向量是单位向量。网页会自动单位化输入向量，并明确显示单位方向 `u`，可以帮助学生避免把普通方向向量直接代入造成错误。

### 5. 视觉清晰，便于观察

网页提供亮色 / 暗色主题、三维曲面透明显示、线框辅助、坐标网格和高对比箭头，使曲面结构和关键对象更容易看清。

## 七、测试结果

### 1. 典型例子测试

测试函数：

```text
z = x^2 + y^2
```

测试点：

```text
P0 = (1, 1)
```

测试方向：

```text
v = (1, 0)
```

网页显示结果：

```text
P0 = (1, 1, 2)
∇f(P0) = (2, 2)
u = (1, 0)
D_u f(P0) = 2
```

人工计算结果：

```text
fx = 2x
fy = 2y
∇f(1,1) = (2,2)
D_u f(P0) = (2,2) · (1,0) = 2
```

结论：网页计算结果与理论结果一致。

### 2. 公式输入测试

以下函数均能正常输入并生成图像：

```text
x^2 + y^2
x^2 - y^2
sin(x)cos(y)
sqrt(x^2 + y^2)
ln(x^2 + y^2 + 1)
```

结论：网页能够支持常见初等函数的输入与展示。

## 八、使用注意事项

1. 当前网页主要支持显式曲面 `z=f(x,y)`，不支持隐式曲面。
2. 点只需要输入 `(x0,y0)`，网页会自动计算对应的 `z0`。
3. 方向向量只需要输入二维方向 `(vx,vy)`，网页会自动单位化。
4. 如果函数在某点无定义，方向导数无法计算，网页会给出提示。
5. 建议课堂展示时优先使用预设函数，便于快速说明概念。
